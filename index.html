Como fazer o alistamento militar?

Requisitos para fazer o alistamento


  • Sexo: O alistamento militar é obrigatório para os cidadãos do sexo masculino.
  • Idade: O alistamento militar deve ser feito no ano em que o indivíduo completa 18 anos.
  • Nacionalidade: O alistamento militar é obrigatório para cidadãos brasileiros natos ou naturalizados.

    • Documentação para fazer o alistamento


  • Documento de identidade (RG) ou equivalente.
  • Cadastro de Pessoa Física (CPF).
  • Certidão de nascimento ou equivalente.

    • O que fazer agora?

    1. Acesse o site oficial: Acesse o site oficial do alistamento militar online no Brasil.
    2. Preencha o formulário: No site, preencha o formulário de alistamento com suas informações pessoais
    3. Agende a data de apresentação: Após selecionar a forma de apresentação, você poderá agendar uma data para comparecer à Junta de Serviço Militar mais próxima.
    4. Compareça à Junta de Serviço Militar: No dia agendado, vá até a Junta de Serviço Militar com os documentos solicitados
